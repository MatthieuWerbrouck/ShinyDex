# ShinyDex

Tracker de shinies Pokémon avec recommandation de méthode de chasse et projets collaboratifs.

## Stack

- React 19 + Vite 8
- Tailwind CSS v4
- ESLint + Prettier

## Lancer le projet

```bash
npm install
npm run dev
```

## Scripts

| Commande | Description |
|---|---|
| `npm run dev` | Serveur de développement |
| `npm run build` | Build production |
| `npm run lint` | Lint ESLint + Prettier |
| `npm run preview` | Prévisualiser le build |
